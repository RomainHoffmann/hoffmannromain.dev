# hoffmannromain.dev

Portfolio — tuiles projets, expand plein écran, galerie screens.

## Dev

```bash
npm install
npm run dev
```

Ouvre [http://localhost:3000](http://localhost:3000).

## Contenu

- Projets : `src/data/projects.ts`
- Screens : `public/projects/screens/{slug}/` (png, jpg, webp — nombre variable)
- Build : `npm run build`
